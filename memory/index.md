# Memory Vault -- Router

Entry point to the Obsidian-style memory vault. Read this first, then follow at most 2
wikilinks (see the memory-protocol skill). Machine bookkeeping lives in `state/`, not here.

## Folders
- `daily/` -- dated session notes (tier: trace, scope: session).
- `projects/` -- distilled per-project notes.
- `decisions/` -- durable decisions (append-only; supersede, never overwrite).
- `entities/` -- people, systems, and accounts referenced across notes.
