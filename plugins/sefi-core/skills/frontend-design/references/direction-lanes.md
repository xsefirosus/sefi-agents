# Direction Lanes -- named starting points for the direction-first rule

Name one of these lanes, or name an original direction with the same rigor: a
fingerprint, a rationale tied to the audience, and what it explicitly avoids. This
list is vocabulary, not a cage. The default lane is restrained; immersive/maximalist
is opt-in only.

## 1. Restrained (default)
Fingerprint: one accent doing real work, quiet neutral surroundings, generous
whitespace, hierarchy carried by type and spacing.
Tendency: neutral surfaces with a single saturated accent; a workhorse sans with a
real weight range.
Not for: briefs that ask for spectacle or an editorial voice.

## 2. Editorial
Fingerprint: serif display over readable columns, hairline dividers, a reading rhythm
like a well-set publication.
Tendency: ink-on-paper contrast, restrained warm neutrals; serif display with sans or
serif body.
Not for: dense operational tools where scanning beats reading.

## 3. Brutalist
Fingerprint: exposed structure, hard edges, high contrast, oversized type, minimal
ornament -- unpolished on purpose.
Tendency: near-monochrome plus one loud accent; grotesque sans or mono, tight leading.
Not for: trust-sensitive domains (finance, health) or long-form reading.

## 4. Technical / Mono
Fingerprint: monospace accents, grid-locked layout, terminal-inspired restraint,
data-first.
Tendency: dark-friendly neutrals with a phosphor-style accent used sparingly; mono for
data and labels, sans for prose.
Not for: consumer lifestyle products or emotional storytelling.

## 5. Organic / Soft
Fingerprint: rounded forms, soft shadows, gentle motion, warmth over precision.
Tendency: warm neutrals and muted naturals; humanist sans with a generous x-height.
Not for: dense data tools; softness at density reads as mush.

## 6. Structured / Data-dense
Fingerprint: table-first, high information density, small type set carefully, minimal
decoration -- every pixel is information.
Tendency: cool neutrals with semantic color reserved for state; compact sans with
tabular figures.
Not for: marketing or first-touch surfaces; density before trust reads hostile.

## 7. Playful
Fingerprint: one saturated accent used bravely, friendly rounded type, sparing
illustrative touches. Easy to slide into slop -- use deliberately.
Tendency: a bright accent on calm ground; rounded sans with personality.
Not for: professional or regulated contexts, or as an unexamined default for consumer
apps.

## 8. Luxury / Quiet
Fingerprint: deep neutral palette, generous negative space, refined details, nothing
shouting.
Tendency: near-black, bone, at most one metallic-adjacent accent; refined serif or
light-weight sans with wide tracking on labels.
Not for: information-dense or utilitarian tools; quiet costs density.

## 9. Immersive / Maximalist (opt-in only)
Fingerprint: full-bleed imagery, scroll-driven motion, layered depth, spectacle as the
point. Only when the brief explicitly asks for it.
Tendency: high-contrast, imagery-led color; oversized display type.
Guardrails: animate only transform and opacity; use will-change deliberately; always
respect prefers-reduced-motion; degrade hover-dependent effects on touch devices.
Not for: anything a user operates daily; spectacle fatigues.

## 10. Archival / Print
Fingerprint: print-inspired grid, numbered references, captions and footnotes as
first-class elements, color cued through tone rather than literal texture.
Tendency: paper-tone neutrals with archival red or blue as the single accent; serif
body with mono annotations.
Not for: fast task flows; this register rewards lingering, not doing.

## Custom mode (the escape hatch)
A brief that fits none of these names an original direction with the same rigor:
fingerprint, palette and type tendency, what it avoids, and a one-line audience
rationale. An original direction is held to the same anti-slop checklist as any lane.
